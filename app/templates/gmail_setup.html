Gmail Integration Setup

Follow these steps to integrate your Email Prioritizer with Gmail:

Step 1: Create a Google Cloud Project
  1. Go to the Google Cloud Console
  2. Create a new project (or select an existing one)
  3. Navigate to "APIs & Services" > "Library"
  4. Search for "Gmail API" and enable it
Step 2: Create OAuth Credentials
  1. In your Google Cloud project, go to "APIs & Services" > "Credentials"
  2. Click "Create Credentials" and select "OAuth client ID"
  3. Select "Desktop application" as the application type
  4. Name your client (e.g., "Email Prioritizer")
  5. Click "Create"
  6. Download the JSON file with your credentials
Step 3: Configure Email Prioritizer
  1. Rename the downloaded JSON file to "gmail_credentials.json"
  2. Place this file in the "data" folder of your Email Prioritizer installation
  3. Return to the dashboard and click "Connect Gmail"
  4. Follow the authentication prompts

What This Integration Does

Once connected, the Email Prioritizer will:

  • Create "Priority" labels in your Gmail account
  • Apply appropriate priority labels to your emails
  • Allow you to search and filter emails by priority in Gmail

Gmail's native priority system (Primary, Social, Promotions tabs) will still function separately.
